# API

## 一、 API 文档

> 支持: mermaid、echarts

### type

描述: 预览类型
类似: String
默认值: html
可选值: html mind

### theme

描述: 设置主题
类似: String
默认值: light
可选值: light

### children

描述: props.children
类型: Object
默认值: ---

### options

描述: 同 markdown-to-jsx 剧透参考该 npm 包
类型: Object
默认值: ---

### style

描述: 为组件追加 style
类型: Object
默认值: ---

### className

描述: 为组件追加 className
类型: String
默认值: ---
